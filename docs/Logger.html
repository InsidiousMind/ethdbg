Class: Logger

Logger()

This allows messages during runtime to be displayed to the programmer with various levels of detail. The higher the level, the more information you receive during runtime. Log Levels 5 - Everything you never wanted to know about the program is displayed. 4 - Warnings and anything worse. 3 - General errors and all below. 2 - critical errors and higher. 1 - Only fatal errors.
